Brackets in Javascript

Parenthesis () uses:

  • Calling or defining a function
    • function doSomething(arguments) { commands; return ...; }
    • result = doSomething(...)
  • evaluating boolean conditions
    • while (condition) {...}
    • if (condition) {...}
  • Giving priority in mathematical operations: (index+1)%index

Square bracket [] uses

  • defining, accessing and modifying arrays: array[0] = item1
  • accessing and modifying/creating javascript object properties: object['propertyName'] = property

Curly bracket {} uses

  • function definitions: function(...) {commands}
  • code inside conditional tests and loops:
    • if (...) {commands}
    • while (...) {commands}
  • Defining objects: object = {property: value, ...}

Quotation Marks " '

Single and double quotation marks can be used interchangably in javascript, they both get used to define strings. It's possible to use single quotation mark inside a string bounded by double marks or vice versa and have them not treated as special characters: "It's great" and 'This is a "string"'

To nest the same kind of quote marks inside a string you have to use a backslash (escape character) to request they not get treated as special characters "He said 'She said \"He said\"'".
